# Отладка программ

Используя отладчик и чтение кода найти ошибки. Отладчик: gdb (в VSCode или консольный)

| Оценка | Требования |
|--------|------------|
| **3** | Найдены и исправлены ошибки в ЛЮБЫХ 2-х файлах из task1-task4 |
| **4** | Найдены и исправлены ошибки во ВСЕХ файлах task1, task2, task3, task4 |
| **5** | Исправлены все файлы task1-task4 + task5 |
| **\*** | Исправлены все ошибки в task_star.c (ASan/valgrind = 0 ошибок) |

## Инструменты

- gdb (консольный / VSCode)
- ASan / valgrind
