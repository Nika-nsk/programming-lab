# Курсовой проект: Визуальный Фитнес-Дневник на Go (MVP)

 Веб-приложение (бэкенд на Go + минималистичный фронтенд) для планирования тренировок по дням и отслеживания прогресса выполнения упражнений в реальном времени.

## Реализованный функционал (MVP)
- **Управление тренировочными днями**: Создание и удаление тренировочных дней (например, "День ног", "Спина/Бицепс").
- **Управление упражнениями**: Добавление упражнений внутрь конкретного дня с указанием подходов и повторений, а также их удаление.
- **Интерактивный трекинг**: Возможность отметить упражнение выполненным (с визуальным зачеркиванием в интерфейсе).

## Архитектура проекта
Приложение строго разделено на слои (пакеты) согласно техническому заданию:
1. `cmd/app/main.go` — Точка входа, запуск сервера и раздача веб-интерфейса.
2. `cmd/app/index.html` — Визуальный интерфейс (HTML/CSS/JavaScript).
3. `internal/config/` — Слой конфигурации (чтение настроек среды).
4. `internal/handler/` — HTTP REST API слой (обработка запросов, маршрутизация, JSON).
5. `internal/service/` — Слой бизнес-логики (проверка валидности данных и ограничений).
6. `internal/repository/` — Слой хранения данных (In-Memory база данных на безопасных `map` и срезах с блокировками `sync.RWMutex`).
7. `internal/model/` — Описание структур данных Go.

## Инструкция по запуску

### 1. Запуск Unit-тестов
Для проверки бизнес-логики встроенными средствами тестирования Go выполните:
```bash
go test ./... -v
```

### 2. Запуск веб-приложения
Запустите сервер командой из корневой папки:
```bash
go run cmd/app/main.go
```

После запуска откройте любой браузер и перейдите по адресу:
👉 **http://localhost:8080**
